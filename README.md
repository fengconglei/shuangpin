# 双拼练习

一个用于练习小鹤双拼的网页应用。项目采用纯前端实现，打开 `index.html` 即可使用，不需要后端服务、不需要数据库，也不需要安装依赖。

## 项目简介

这个项目的目标是帮助用户通过分阶段练习，逐步掌握小鹤双拼的声母、韵母、音节和常用汉字输入。

页面采用深色玻璃拟态风格，内置键位提示、虚拟键盘、计分系统、连击系统、练习进度和速度挑战模式，适合双拼入门、键位记忆和日常打字训练。

## 功能特点

### 1. 小鹤双拼键位映射

项目内置小鹤双拼规则，包括：

- 声母键位
- 韵母键位
- 特殊声母映射
- 零声母规则
- 常用音节与词语练习数据

特殊声母规则：

```text
zh → v
ch → i
sh → u
```

### 2. 五个练习阶段

项目分为 5 个练习模块：

| 阶段 | 名称 | 说明 |
| --- | --- | --- |
| 1 | 声母练习 | 学习声母与键位的对应关系，包括 zh/ch/sh 等特殊声母 |
| 2 | 韵母练习 | 掌握所有韵母的键位映射 |
| 3 | 音节练习 | 输入完整音节的两个按键：声母 + 韵母 |
| 4 | 词语练习 | 看汉字打双拼，巩固常用汉字输入 |
| 5 | 速度挑战 | 限时 60 秒，随机输入汉字，挑战最高分 |

### 3. 虚拟键盘提示

页面内置虚拟键盘，按键上会显示对应韵母或特殊声母，帮助用户快速建立键位记忆。

### 4. 自动判题与反馈

输入后系统会自动判断对错，并给出正确、错误、提示高亮等视觉反馈。

### 5. 得分与连击机制

答对题目会获得分数，连续答对会增加连击分；系统会记录当前得分、连击数和最长连击。

### 6. 本地进度保存

项目使用浏览器 `localStorage` 保存各阶段最高分。刷新页面或重新打开后，仍可看到历史最高成绩。

### 7. 速练模式

速度挑战模式为 60 秒限时练习，适合在熟悉规则后训练反应速度和输入熟练度。


`index.html` 中包含：

- HTML 页面结构
- CSS 样式
- 小鹤双拼规则数据
- 练习题库数据
- 练习阶段配置
- 键盘监听与判题逻辑
- 分数、连击和进度保存逻辑

## 如何运行

### 方法一：直接打开

1. 下载或克隆本项目
2. 找到 `index.html`
3. 用浏览器打开即可使用

### 方法二：使用本地服务器

在项目目录下运行：

```bash
python -m http.server 8000
```

然后在浏览器打开：

```text
http://localhost:8000
```

## 使用说明

1. 打开页面后，选择一个练习阶段
2. 根据页面提示输入对应按键
3. 声母和韵母练习通常只需按 1 个键
4. 音节、词语和速度挑战需要输入 2 个键
5. 输入后系统会自动判断答案
6. 答对后自动进入下一题
7. 答错时会显示正确按键提示
8. 练习结束后查看得分、正确率和最长连击

## 小鹤双拼核心规则

### 特殊声母

```text
zh = v
ch = i
sh = u
```

### 部分常用韵母映射

```text
ai  = d
an  = j
ang = h
ao  = c
ei  = w
en  = f
eng = g
ing = k
iong / ong = s
ian = m
iang / uang = l
iao = n
ie  = p
iu  = q
uan = r
üe  = t
un  = y
ou  = z
```

### 示例

```text
中：zhōng → zh + ong → v + s → vs
国：guó   → g + uo  → g + o → go
人：rén   → r + en  → r + f → rf
学：xué   → x + üe  → x + t → xt
双：shuāng → sh + uang → u + l → ul
拼：pīn   → p + in  → p + b → pb
```
